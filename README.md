#  Paquexpress - Sistema de Gestión de Entregas

Sistema completo para gestión de entregas con aplicación móvil en Flutter y API backend en FastAPI.

## Características

- **App Móvil**: Registro de entregas con cámara y GPS
- **API REST**: Backend seguro con FastAPI
- **Base de Datos**: MySQL con historial completo
- **Autenticación**: Sistema seguro de login para repartidores

## Tecnologías Utilizadas

- **Frontend**: Flutter + Dart
- **Backend**: FastAPI + Python
- **Base de Datos**: MySQL

## Prerrequisitos

Antes de comenzar, asegúrate de tener instalado:

- **Flutter** 3.0 o superior
- **Python** 3.8 o superior
- **MySQL** 8.0 o superior
- **Git**
